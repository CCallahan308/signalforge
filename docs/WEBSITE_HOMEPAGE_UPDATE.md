## FEATURED PROJECT

### 🏆 SignalForge - Production Churn Intelligence

**The Challenge:** Predict customer churn with statistical rigor and quantify business impact

**The Solution:** Built end-to-end ML system with:
- ✅ **0.850 ± 0.013 AUC** with 5-fold cross-validation
- ✅ **Statistical significance testing** (p<0.01 vs alternatives)
- ✅ **$1.67M annual revenue** at risk identified
- ✅ **1.21x - 1.81x ROI** expected from interventions
- ✅ **Bootstrap confidence intervals** on all metrics
- ✅ **Calibration analysis** for probability accuracy
- ✅ **58 engineered features** with learned weights

**Tech Stack:** Python • PostgreSQL • scikit-learn • Docker • Streamlit
